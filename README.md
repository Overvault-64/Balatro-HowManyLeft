# HowManyLeft

A **Steamodded** mod for **Balatro** that shows, at a glance, how many cards sharing the hovered card's suit or rank can still be drawn from the deck.

It adds two useful lines to standard playing card tooltips:

- how many cards of the **same rank** are left in the deck;
- how many cards of the **same suit** are left in the deck.

Tooltip text uses the game's localization when available.
